=KR=

1. "Bandage_package"를 Scene에 드래그하여 넣어줍니다.
2. unpack prefab을 합니다.
3. "Bandage"를 아바타 바로아래에 넣습니다.
4. "Bandage_position"를 원하는 위치(머리,팔,다리등)로 이동하고 부착할 본 아래에 놓습니다.
반창고의 쉐이프키를 이용하여 적당한 굴곡을 만들어 줍니다.
5. 빈 "Bandage_package"를 지워줍니다
6. "Avatars 3.0 Manager" 툴을 이용하여 기존 FX애니메이터와 "Bandage_FX_animator" 를 합쳐줍니다

=EN=
1. Drag "Bandage_package" into the scene.
2. unpack prefab
3. Put "Bandage" right under the avatar.
4. Move the "Bandage_position" to the desired position (head, arms, legs, etc.) and place it under the bone to attach.
Use the shape key of the bandage to make a proper curve.
5. Erase the blank "Bandage_package"
6. Use the Avatars 3.0 Manager tool to merge the existing FX animator and Bandage_FX_animator

=JP=
1. "Bandage_package"をSceneにドラッグして入れます。
2. unpackprefabをします。
3. "Bandage"をアバターの真下に入れます。
4. 「Bandage_position」を希望する位置(頭、腕、脚など)に移動し、付着するBoneの下に置きます。
シェイプキーを使って適度なカーブを作ります
5. 空っぽの"Bandage_package"を消します。
6. "Avatars3.0Manager"ツールを利用して既存のFXアニメーターと"Bandage_FX_animator"を併合します。
